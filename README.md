# pypmsi

Lire les fichiers du PMSI avec python / pola.rs


## Installation


```sh
git clone https://github.com/GuillaumePressiat/pypmsi.git
poetry install
```

avec pip

```sh
pip install https://github.com/GuillaumePressiat/pypmsi/releases/latest/download/pypmsi-0.1.0-py3-none-any.whl
```

## Exemple


```python
import polars
import pypmsi as pm
```


SpГ©cifier les paramГЁtres dans la fonction

```python
rsa = pm.irsa(290000017, 2021,5, '~/Documents/data/mco', typi = 4)
rsa
```

ou bien, dГ©finir un noyau de paramГЁtres

```python
p = pm.noyau_pmsi(finess = 290000017, annee = 2021, mois = 5, path = '~/Documents/data/mco')
rsa = p.irsa()
rsa
```

ou bien, indiquer le chemin du fichier et l'annГ©e, et le lire

```python
mon_rsa = pm.chemin_pmsi(filepath = '~/Documents/data/mco/290000017.2021.5.rsa', annee = 2021)
rsa = mon_rsa.read_rsa()
rsa
```
(du coup le nom du fichier peut-ГӘtre formatГ© diffГ©rement).

On peut modifier en ligne les paramГЁtres, exemple :

```python
p = pm.noyau_pmsi(finess = 290000017, annee = 2021, mois = 12, path = '~/Documents/data/mco')
# lire les donnГ©es 2022
rsa = p.irsa(annee = 2022)
rsa
```
